# Public Assets

This directory contains the static assets for the Jewish Mother's Deli website.

## Required Assets

- `fulllogo_transparent.png` - Main logo image
- `jewish-mothers-deli-favicon.png` - Favicon image

## Asset Sources

These assets should be copied from the original `coming-soon-pages/templates/jewish-mothers-deli/` directory.

## Usage

The assets are referenced in the React components and will be served from the root path (e.g., `/fulllogo_transparent.png`).
